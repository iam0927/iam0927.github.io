数组的高阶方法

发表于 2018-01-28 编辑于 2021-07-20 分类于 技术分享

函数

  • 函数可以当做参数传递
  • 可以作为返回值

    之所以称作高阶函数，主要原因在于很多其他语言不支持或者是支持起来很困难

forEach

输出数组的每一项和下标 forEach不能终止或者跳出循环

some

根据一个条件，验证数组中是否存在符合该条件的元素，符合返回true,不符合返回false

验证是否含有 苹果

every

every和some对应，验证数组是否都满足某些条件 跟some正好相反
验证数组是否都满足我的条件，都符合返回true，有一项不符合则返回false

find、findIndex

find和findIndex都为es6方法
find找到数组中符合该条件的第一个元素
findIndex找到数组中符合该条件的第一个元素的下标

filter

找到所有符合该条件的元素放到一个数组中去作为结果
如果没有找到则返回一个空数组

map

对数组中每一项进行操作，返回到一个新数组中

includes

验证数组是否有符合给定值得元素，需要传递一个具体的值,返回一个布尔值，同样也是es6方法
和indexOf几乎是一个东西
some是指定具体详细条件，而includes是判定指定元素

reduce

reduce方法用来做累积

reduce参数有长度限制，不同的长度结果是不一样的

  • 第1个参数 直接返回第1 个
  • 第二个参数用来指定初始化的值，也就是prev的初始值，当没有指定的时候，prev的初始值是数组的第[0]项元素
